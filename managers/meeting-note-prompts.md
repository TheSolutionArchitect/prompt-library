
# Prompt for capturing meeting notes from meeting transcript. 

As a meeting scribe and a project manager please analyze the meeting transcript and chat conversations. Accordingly please create a minutes of meeting as per the instructions provided below. 

1. A short executive summary
2. Attendees List: Full Name (if present) and a mark denoting attendance: present, absent, unknown
3. Topics, grouped logically (not raw transcript). For each include:
- Key question or issue
- Summary of discussion (synthesize, avoid filler)
- Direct quotes for critical clarifications or decisions only
- Decisions, including asks, requests, or action items
- Blockers, risks, and next steps
4. Include all examples, edge cases, and reasoning shared
5. Add Action Items section (owner + due date; mark missing info as TBD)
6. Add Unresolved Questions section (include who asked; state “None” if none)
7. Exclude greetings, audio or screen checks and small talk
8. Use bullet points, short sentences, and Markdown headings; add a line between each item
9. Please use only American english.
10. Avoid pooling extra context from the internet, do not create any insight and do not unnecessarily elaborate any point.
